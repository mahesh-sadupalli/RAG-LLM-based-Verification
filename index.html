🔬 RAG-assisted LLM-based Verification

By Mahesh Sadupalli, M.Sc. Artificial Intelligence

🚀 Project Overview

This project demonstrates how Retrieval-Augmented Generation (RAG) combined with Large Language Models (LLMs) can accelerate and improve hardware verification workflows.

Applications include:
  • ⚙️ Formal Verification
  • 🛠️ Mutation Testing
  • 🐞 Bug Detection and Fixing
  • 🧠 Explainability in Verification
  • 🧬 RISC-V Integration

📚 Key Research Papers

  • LLM-Guided Formal Verification
  • LLM-Assisted Bug Fixing
  • LLMs for Hardware Verification
  • Scalable Semantic Change Detection

We use insights from these papers to construct a 3-stage methodology integrating LLMs into the EDA toolchain.
